## PROJECT OVERVIEW

Build **ChabadLabs.ai** — a futuristic, dark-themed resource hub for Chabad shluchim to discover and deploy AI tools. The aesthetic is inspired by **dynexcoin.org**: deep dark backgrounds, bold oversized typography, gold glowing accents, generous whitespace, and a premium tech feel. Think Vercel meets crypto meets mission-driven community.

**Tagline: "Welcome to the future of shlichus."**

## TECH STACK

- **Framework**: Next.js 14+ with App Router (`/app` directory)
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui (dark theme)
- **Icons**: Lucide React
- **Fonts**: Use `Inter` for body and `Space Grotesk` or `Sora` for headings (import from Google Fonts)
- **Data**: Static JSON files in `/data` (no database for MVP)
- **Chat**: Mock AI chat (no API key needed — hardcoded responses)
- **Deployment**: Replit

### Design Rules

- **Dark everywhere.** Background is #06060A. No white backgrounds anywhere.
- **Gold is the star.** Primary buttons, accent borders, the chat bubble, hover glows — all gold.
- **Typography is bold and oversized.** Hero headline should be 64-80px. Section titles 40-48px. Let text breathe.
- **Numbered section labels** like Dynex uses: small monospaced gold text above sections (e.g., "001 — RESOURCES", "002 — WEBINARS", "003 — GET STARTED").
- **Cards have subtle glowing borders.** Use `border border-[#1E1E2A] hover:border-[#C49A2A]/30` with a subtle `shadow-[0_0_30px_rgba(196,154,42,0.08)]` on hover.
- **Generous spacing.** Sections should have 120-160px vertical padding. Cards should breathe. Don't crowd.
- **Bento grid** on the landing page for feature showcase — asymmetric card sizes like Linear/Vercel dashboards.
- **Smooth transitions.** All hover states transition 300ms. Cards lift slightly on hover.
- **No images or stock photos.** Use Lucide icons, abstract gradients, or subtle grid patterns for visual interest.
- **Mobile-first.** Everything must look great on phones. Bento grid collapses to single column.

### Subtle Background Effects

- Add a faint radial gold gradient glow behind the hero section (centered, 40% opacity, very large radius)
- Optional: a subtle CSS grid pattern overlay on the background (thin lines at ~5% opacity) to give depth — like graph paper in the dark

## CRITICAL NOTES

1. **Dark theme ONLY.** Configure shadcn/ui with dark mode. Set `<html className="dark">` in the root layout. Background is always #06060A.
2. **Gold (#C49A2A) is the primary accent everywhere.** Buttons, hover states, borders, badges, the chat bubble, links on hover.
3. **No images or stock photos.** Icons only (Lucide React). Visual interest comes from the layout, spacing, glows, and typography.
4. **Chat widget on ALL pages** — rendered in the root layout as a fixed-position element.
5. **All external links open in new tabs** with `target="_blank" rel="noopener noreferrer"`.
6. **Mobile-first responsive.** Many shluchim will access from phones.
7. **Section labels** are a consistent pattern: monospaced, uppercase, gold, small (12-14px), with a number prefix (001, 002, etc.).
8. **Keep code clean and modular.** This codebase will be handed to Claude Code for the next iteration. Components should be self-contained and well-named.
9. **Smooth animations.** Cards: `transition-all duration-300`. Hover: translate-y-[-2px]. Chat panel: slide-up animation.
10. **No authentication, database, or payments.** This is a static MVP — all data from JSON files.
